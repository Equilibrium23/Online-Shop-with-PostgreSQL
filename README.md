## App Description
#### My app (shop with monitors) is created for two types of users :
  - #####  shop clients are able to:
    - ###### login
    - ###### search and filters monitors 
    - ###### make an order
    - ###### add an opinion
    - ###### submit a refund 
    - ###### add and set main adress
  - ##### shop admins are able to:
    - ###### login
    - ###### explore database
    - ###### add/update/delete some records
